Echando un vistazo a Less, la otra cara del smarter CSS

Por Ricard Molina Ferret

LESS es un super contexto de CSS que pretende dotarlo de un comportamiento dinámico a través de variables, mixins, operaciones y funciones. Para utilizar LESS no necesitas un plug-in ni nada especial. La idea detrás de la herramienta es que el navegador interpreta codigo CSS normal y corriente producido a través de un proceso de compilacion de nuestras plantillas escritas en LESS.

Lo único que cambia respecto al CSS es que las plantillas LESS deben ser compiladas. Si usamos un servidor Web basado en Node.js o bien servimos nuestras hojas de estilo CSS por medio de un servidor estatico, podemos compilar nuestras plantillas LESS en tiempo de ejecucion (del servidor) utilizando para ello Node.

Las variables nos permiten definir en un solo lugar un valor y luego usarla en múltiples sitios de nuestro codigo CSS. Esto nos permite maximizar la reutilizacion de codigo. Asi, si tenemos que modificar una variable, solo tendremos que cambiarlo solo en un lugar de nuestro archivo CSS.

Para finalizar, mostramos un fragmento de codigo CSS escrito en LESS donde se puede ver la anidacion de reglas: un recurso muy valioso para la ordenacion y la mantenibilidad del codigo.
